## 2. 增加信息类型

**重要说明**：根据小程序实际功能，**不需要添加用户信息类型**。

**原因：**
- 小程序本身不收集用户个人信息
- 活动报名通过跳转到金数据小程序完成，个人信息收集在金数据小程序中进行
- 小程序仅使用微信云开发的基础服务（不涉及用户信息收集）
- 已设置 `traceUser: false`，云函数不追踪用户身份

**如果表单要求必须填写信息类型**，可以填写：
```
无
```
或
```
不收集用户信息
```

## 3. 第三方插件信息/SDK信息

**检查结果**：小程序未接入第三方插件/SDK

**说明：**
- 小程序使用微信云开发服务（微信官方服务，无需单独声明）
- 未使用第三方统计、分析、广告等SDK
- 未使用第三方支付、地图等插件
- **金数据小程序跳转**：使用微信官方API `wx.navigateToMiniProgram` 跳转到金数据小程序，不是第三方SDK集成。个人信息收集在金数据小程序中进行，由金数据负责处理。

**如果表单要求必须填写**，可以填写：
```
无第三方插件/SDK
```

**关于金数据小程序跳转的说明：**
- 小程序使用 `wx.navigateToMiniProgram` API 跳转到金数据小程序
- 这是微信官方提供的API，不是第三方SDK
- 跳转后，用户填写表单和提交信息都在金数据小程序中完成
- 个人信息收集和处理由金数据负责，遵循金数据的隐私政策
- **不需要在小程序隐私保护指引中声明**，因为小程序本身不收集这些信息
